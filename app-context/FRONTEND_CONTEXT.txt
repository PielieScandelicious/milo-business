OVERVIEW
--------------------------------------------------------------------------------
Milo is an iOS mobile application designed to help users track and
analyze their spending through receipt scanning and AI-powered insights. The
app features a friendly AI
assistant named "Milo".

Primary Purpose: Personal finance and spending analysis app combining automated
receipt scanning, visual analytics, and AI-driven recommendations.


APPLICATION TYPE
--------------------------------------------------------------------------------
- Platform: iOS (native)
- Framework: SwiftUI
- Architecture: MVVM (Model-View-ViewModel)
- Bundle ID: com.deepmaind.Milo
- Firebase Project ID: dobby-dd70e


TECH STACK
--------------------------------------------------------------------------------

Frontend (iOS):
- SwiftUI (modern declarative UI framework)
- Swift programming language
- StoreKit 2 (in-app purchases and subscriptions)

Authentication:
- Firebase Authentication
- Google Sign-In integration
- Keychain for secure token storage

Backend:
- Railway-hosted REST API
- Production: https://scandalicious-api-production.up.railway.app
- Non-Prod: https://scandalicious-api-non-prod.up.railway.app
- API Version: v2

External Services:
- Firebase (Authentication, Storage)
- Google Cloud Platform
- Railway (API hosting)


CORE FEATURES
--------------------------------------------------------------------------------

1. RECEIPT SCANNING & MANAGEMENT
   - Camera-based receipt capture with quality checking
   - Photo library import support
   - Share Extension for uploading receipts from other apps
   - Multipart form data upload to backend
   - Duplicate detection with confidence scores
   - Receipt details with transaction breakdown
   - Rate limiting (monthly per-user limits)
   - Health score averaging for food items

2. ANALYTICS & DASHBOARD
   - Spending trends by period (week/month/year)
   - Category breakdown with 16+ categories
   - Store-level analytics and spending breakdown
   - Visual charts: donut charts, liquid gauges
   - Transaction filtering by store, period, and category
   - Summary views with category distribution

3. AI CHAT ASSISTANT ("MILO")
   - Streaming chat interface using Server-Sent Events (SSE)
   - Markdown rendering with table support
   - Conversation history management
   - Sample prompt cards for user guidance
   - Backend-powered AI responses

4. USER MANAGEMENT
   - Google Sign-In authentication
   - Profile completion flow for new users
   - Onboarding experience
   - User preferences (name, gender)
   - Token management for share extension

5. SUBSCRIPTION SYSTEM
   - In-app purchases (Monthly/Yearly plans)
   - Premium feature gating
   - Paywall system
   - Subscription status tracking

APP NAVIGATION STRUCTURE
--------------------------------------------------------------------------------

Tab 1: VIEW (chart.pie.fill icon)

├── Receipt history
├── Store breakdowns
└── Spending trends

Tab 2: SCAN (qrcode.viewfinder icon)
├── Custom camera view for receipt capture
├── Receipt upload functionality
├── Upload success confirmation
├── Recent receipt details
└── Saved receipts list

Tab 3: MILO (sparkles icon)
├── AI chat assistant interface
├── Spending insights and recommendations
├── Markdown-formatted responses
└── Conversation history
