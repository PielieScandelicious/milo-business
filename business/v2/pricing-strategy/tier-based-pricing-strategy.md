| Metric | Gold Tier | Platinum Tier | Blended |
|:-------|----------:|--------------:|--------:|
| **Monthly Price** | $5.99 | $12.99 | - |

> **Conservative Design Philosophy:** This pricing model prioritizes sustainable unit economics over aggressive growth. ## 2. Pricing Philosophy

</div>

### Core Principles

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                       PRICING DESIGN PRINCIPLES                             │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│  1. SUSTAINABLE UNIT ECONOMICS                                              │
│     Every tier must achieve positive gross margin from day one              │
│     No "land grab" pricing that sacrifices profitability                    │
│                                                                             │
│  2. CLEAR VALUE DIFFERENTIATION                                             │
│     Each tier upgrade must deliver tangible, measurable value               │
│     Pricing gaps reflect genuine feature/capacity differences               │
│                                                                             │
│  3. CONVERSION-OPTIMIZED TRIAL                                              │
│     14-day trial creates urgency without permanent free users               │
│     10 receipts enough to demonstrate value, not enough to satisfy          │
│                                                                             │
│  4. CONSERVATIVE PROJECTIONS                                                │
│     All forecasts use pessimistic assumptions                               │
│     Better to exceed conservative targets than miss optimistic ones         │
│                                                                             │
│  5. MARGIN PROTECTION                                                       │
│     Receipt limits calibrated to protect 65%+ gross margins                 │
│     Buffer for Veryfi cost increases built into pricing                     │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

### Why No Free Tier?

| Consideration | Impact |
|:--------------|:-------|
| **Variable Cost Reality** | Each free user costs $0.80+ in OCR fees |
| **Conversion Psychology** | Trial creates urgency; free tier enables indefinite delay |
| **Market Positioning** | Premium positioning conflicts with free tier |
| **B2B Data Quality** | Paying users provide higher-quality, more consistent data |
| **Support Burden** | Free users consume support without revenue contribution |

**Decision:** 14-day free trial replaces permanent free tier. Users must convert or leave, ensuring all retained users contribute revenue.

## 4. Free Trial (14 Days)

</div>

### Trial Design

| Attribute | Specification |
|:----------|:--------------|
| **Duration** | 14 calendar days from registration |
| **Receipt Limit** | 10 receipts total (not renewable) |
| **Chat Limit** | 25 messages total |
| **Card Required** | No (reduces friction) |
| **Auto-Convert** | No (explicit opt-in required) |

### Features Included in Trial

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                          FREE TRIAL FEATURES                                │
├─────────────────────────────────────────────────────────────────────────────┤

│  ━━━━━━━━━━━                          ━━━━━━━━━━━━━━━━━━━━━━━━━            │
│                                                                             │
│  • Receipt scanning (10 max)          • Spending trends over time          │
│  • Health scores (0-5 scale)          • Full category analytics            │
│  • 16 spending categories             • Historical data beyond trial       │
│  • Basic monthly summary              • CSV/PDF exports                    │
│  • AI chat assistant (25 msgs)        • Priority processing                │
│  • Store-level view                   • Custom categories                  │
│                                       • API access                         │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

### Trial Cost Analysis

| Component | Cost |
|:----------|-----:|
| Receipts: 10 × $0.08 (Veryfi OCR) | $0.80 |
| Chat: 25 × $0.0003 (Gemini) | $0.01 |
| **Total Trial Cost** | **$0.81** |

### Trial-to-Paid Conversion Strategy

| Day | Trigger | Action |
|:---:|:--------|:-------|
| 1 | Registration | Welcome email with trial overview |
| 3 | First receipt scanned | "Great start!" + feature highlight |
| 7 | 50% of trial elapsed | Mid-trial value summary |
| 10 | 70% of receipts used | "Running low" notification |
| 12 | 2 days remaining | "Trial ending soon" + discount offer |
| 14 | Trial expires | Final conversion prompt |

### Conservative Conversion Assumptions

| Scenario | Trial-to-Paid Rate | Gold % | Platinum % |
|:---------|:------------------:|:------:|:----------:|
| **Conservative** | 15% | 80% | 20% |
| Base | 20% | 75% | 25% |
| Optimistic | 25% | 70% | 30% |

> **Planning Assumption:** All projections use the conservative 15% conversion rate.

## 11. Risk Analysis

</div>

### Pricing Risks

| Risk | Likelihood | Impact | Mitigation |
|:-----|:----------:|:------:|:-----------|
| Trial conversion below 15% | Medium | High | A/B test conversion flows, adjust trial length |
| Gold churn above 10% | Medium | High | Engagement features, win-back campaigns |
| Competitive price pressure | Medium | Medium | Differentiate on health scores, data quality |
| Veryfi cost increase | Low | High | Negotiate volume contract, evaluate alternatives |
| Apple commission increase | Low | Medium | Annual pricing emphasis, potential Android |
