## Fetch Rewards

### Company Profile

| Attribute | Value |
|:----------|:------|
| **Users** | 12.5M MAU |
| **Valuation** | $2.5B (2022) |
| **Funding** | $440M+ raised |
| **Revenue Model** | Brand partnerships, data licensing |

### Strengths & Weaknesses

| Strengths | Weaknesses |
|:----------|:-----------|
| ✓ Massive user base | ✗ No health features |
| ✓ Strong brand partnerships | ✗ Legacy technology |
| ✓ Established data business | ✗ Privacy concerns |
| ✓ Android + iOS | ✗ No spending analytics |

### Competitive Threat: 🟡 MEDIUM

**Rationale:** Large user base and receipt scanning capability, but different value proposition (rewards vs. insights). Could add health features but would require significant pivot.

| Strengths | Weaknesses |
|:----------|:-----------|
| ✓ Largest cashback app | ✗ Commoditized model |
| ✓ Retail partnerships | ✗ No analytics depth |
| ✓ IPO trajectory | ✗ No health dimension |
| ✓ Strong unit economics | ✗ User value = deals only |

### Competitive Threat: 🟡 MEDIUM

**Rationale:** Similar receipt capability but fundamentally different model. Users seeking insights/health won't find value in cashback-only approach.
