## Key Takeaways

### 1. No Complete Competitor Exists

No single competitor combines:
- Receipt scanning
- Item-level data
- Health scoring
- AI analytics
- Privacy-first design

### 2. Market Validation Abundant

| Signal | Source | Validation |
|--------|--------|------------|
| AI finance demand | Copilot, Cleo | $95-200/yr willingness |
| Health scoring demand | Yuka (80M users) | Massive organic growth |
| B2B data value | NielsenIQ, Circana | $30-75/user pricing |
| Post-Mint opportunity | All finance apps | 12M+ displaced users |

### 3. Partnership Opportunities

| Partner | Type | Value |
|---------|------|-------|
| **Yuka** | Cross-promotion | Health-conscious users |
| **Fetch** | Data complement | Users get both rewards + insights |
| **Copilot** | Integration | Item-level data for finance app |

### 4. Competitive Moats to Build

```diff
+ Health scoring IP (proprietary 0-5 system)
+ Item-level data (unique among finance apps)
+ AI categorization accuracy (95%+)
+ Privacy-compliant data (premium B2B pricing)
+ Brand as "health-aware spending" category leader
```
