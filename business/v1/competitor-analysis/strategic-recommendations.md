# Section 2: Competitive Response Strategy

## Against Receipt Apps (Fetch, Ibotta)

### DO NOT Compete

```diff
- Do not add points/rewards systems
- Do not position as "cashback alternative"
- Do not target deal-seekers/coupon users
- Do not compete on receipt volume incentives
```

### DO Complement

```diff
+ Position as complementary tool
+ "Use Fetch for rewards, MILO for insights"
+ Target users who want MORE than just points
+ Emphasize health value (not monetary value)
```

### Messaging Example

> "Love Fetch for your rewards? Add MILO to understand what you're actually buying. Rewards are great—but knowing your spending is healthier? Priceless."

---

## Against Finance Apps (Copilot, Monarch)

### Key Differentiation

| Dimension | Finance Apps | MILO | Message |
|-----------|--------------|------|---------|
| **Data Source** | Bank transactions | Receipt items | "See what, not just where" |
| **Granularity** | Merchant-level | Item-level | "Every item, not just totals" |
| **Health Insight** | None | Core feature | "Your spending, your health" |
| **Primary User** | Budget-focused | Health-focused | Different psychographic |

### Messaging Example

> "Copilot shows you spent $200 at Walmart. MILO shows you spent $50 on produce (health score: 4.2), $30 on snacks (health score: 1.8), and $120 on household items. See the difference?"

# Section 6: Defensive Strategy

## Building Competitive Moats

### Moat 1: Health Scoring IP

```diff
+ Train proprietary models on health scoring data
+ Build unique item-to-health database
+ Publish research to establish authority
+ File patents on methodology (defensive)
+ Create "MILO Health Score" as recognized metric
```

### Moat 2: Data Network Effects

```diff
+ More users → better AI categorization
+ Better categorization → higher accuracy
+ Higher accuracy → more users
+ Build flywheel before competitors catch up
```

### Moat 3: Brand as Category Leader

```diff
+ Own the "health-aware spending" conversation
+ Become the reference point for the category
+ Build community (Reddit, Discord)
+ Establish thought leadership through content
```

### Moat 4: Privacy Premium

```diff
+ Position privacy as feature, not bug
+ Build trust that enables data monetization
+ Differentiate from data-harvesting competitors
+ Create premium pricing power in B2B
```
