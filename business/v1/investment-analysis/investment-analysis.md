## 1. Company & Product Overview

### Core Product

**Milo** is a native iOS application built with SwiftUI that transforms receipt scanning into actionable financial insights. The platform features:

- ✓ AI-powered OCR via Veryfi API (99.75% accuracy)
- ✓ Proprietary 0-5 health scoring for food items
- ✓ Streaming AI assistant ("Milo") for spending insights
- ✓ Multi-retailer cross-store visibility
- ✓ 16 intelligent spending categories
- ✓ Real-time analytics dashboard

### Technology Stack

| Layer | Technology |
|-------|------------|
| Frontend | Swift / SwiftUI (MVVM) |
| Backend | Python FastAPI 3.11 |
| Database | PostgreSQL (async) |
| AI/ML | Google Gemini 2.0 Flash |
| OCR | Veryfi API v8 |
| Auth | Firebase + Google Sign-In |
| Hosting | Railway (Docker) |
